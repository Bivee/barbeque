# Barbeque

## Status
Currently we are migrating from closed-source version to open-source version and writing documentation.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'barbeque'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install barbeque
```
